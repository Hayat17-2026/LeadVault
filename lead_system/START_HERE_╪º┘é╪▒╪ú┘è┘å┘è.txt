خطوات التشغيل:

1) افتحي الـ folder بـ VS Code

2) بالـ Terminal، الأوامر وحدة وحدة:

   cd lead_system
   python -m venv venv
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
   .\venv\Scripts\Activate.ps1
   pip install flask requests beautifulsoup4 anthropic selenium webdriver-manager google-search-results reportlab

3) ⚠️ حطي المفاتيح:
   - services/search_service.py (سطر 7): الـ SerpAPI key
   - services/email_service.py (سطر 9,10): إيميلك + App Password

4) شغّلي:
   python app.py

5) افتحي: http://127.0.0.1:5000
